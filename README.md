# ESP32 Cricket Scoreboard 64x64

An ESP32-powered Cricket Scoreboard using 64x64 MAX7219 LED Matrix modules.

## Features

- ESP32 DevKit V1
- 64x64 MAX7219 Display (8x8 FC16 modules)
- JSON Score Updates
- REST API
- Web Dashboard
- LittleFS Support
- OTA Updates (Planned)

## Hardware

- ESP32 DevKit V1
- 8 × MAX7219 FC16 Matrix Modules
- 5V Power Supply

## Libraries

- MD_MAX72XX
- ArduinoJson
- ESPAsyncWebServer
- AsyncTCP
